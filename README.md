# Credit-Risk-Insight-Engine

## 🚀 Project Overview

This project presents an interactive Power BI dashboard focused on analyzing loan applications, borrower risk, and default patterns. The goal is to identify high-risk loan segments, understand customer repayment behavior, and support better lending decisions using data-driven insights.

The dashboard helps financial institutions monitor loan performance, reduce default risk, and improve credit assessment strategies.

---

## 🎯 Business Problem

The objective of this project is to analyze loan data and identify:

- High-risk borrowers
- Loan default patterns
- State-wise loan performance
- Loan purpose impact on defaults
- Credit score and debt-to-income trends

This helps in improving loan approval decisions and minimizing financial risk.

---

## 📌 Key Features

- 💰 Total Loan Amount Analysis
- 📄 Loan Application Tracking
- 📉 Default Rate Monitoring
- 🏦 Loan Status Analysis (Current, Default, Late, Paid Off)
- 🌍 State-wise Loan Distribution
- 📦 Loan Purpose Risk Analysis
- 📅 Monthly Loan Trend Analysis
- 🎯 Credit Score & Debt-to-Income Ratio Monitoring

## 💡 Key Insights

### 📌 Loan Performance Insights
- Total loan disbursement exceeded **13M+**, showing strong lending activity
- Average default rate of **24.29%** indicates a significant credit risk area

### 👥 Borrower Risk Insights
- Lower credit score segments show higher chances of default
- High debt-to-income ratio borrowers require stricter approval checks

### 🏷️ Loan Purpose Insights
- **Home Improvement** and **Wedding loans** show the highest default counts
- **Business Loans** and **Medical Expenses** also contribute significantly to defaults

### 🌍 Regional Insights
- States like **MO, IN, and GA** show the highest loan amounts
- Some states show higher concentration of risky borrowers

### 📅 Time-Based Insights
- Loan amount peaks during **April and June**
- Noticeable decline during **September**, indicating seasonal lending behavior

### 📈 Business Recommendations
- Strengthen credit checks for high DTI borrowers
- Focus risk monitoring on high-default loan purposes
- Improve approval strategy using credit score segmentation
- Implement state-wise risk-based lending policies
